外卖骑手配送服务规范

第一章 配送时间规则

1.1 普通订单配送时效
普通外卖订单的标准配送时间为30分钟，从商家接单开始计算。配送时效包含取餐时间和送餐时间，不包含商家出餐等待时间。

1.2 恶劣天气配送延长
遇到以下恶劣天气情况，配送时效自动延长15分钟：
- 中雨及以上（降水量≥10mm/h）
- 大风（风力≥6级）
- 暴雪或道路结冰
- 台风橙色及以上预警
- 能见度低于200米的雾霾天气

恶劣天气延长时效需要系统自动识别，骑手无需手动申报。延长后的总配送时效为45分钟。

1.3 高峰时段配送规则
午餐高峰期（11:00-13:00）和晚餐高峰期（17:00-19:00）期间，系统会适当放宽配送超时判定标准，额外宽限5分钟。高峰时段总配送时效为35分钟。

1.4 超时判定标准
- 配送超时以用户确认收货时间为准
- 如果骑手在配送过程中遇到交通事故、道路封锁等不可抗力因素，需在事发后5分钟内通过APP报备
- 报备成功后，该订单不计入超时统计
- 未报备的异常订单，超时判定从商家出餐完成开始计算

第二章 超时罚款规则

2.1 超时罚款标准
- 超时1-5分钟：不罚款，仅记录
- 超时5-15分钟：扣除配送收入的10%
- 超时15-30分钟：扣除配送收入的20%
- 超时30分钟以上：扣除配送收入的30%，并可能影响骑手等级评分

2.2 罚款申诉流程
骑手如认为超时罚款不合理，可以在收到罚单通知后7个自然日内发起申诉：

第一步：进入APP"我的"→"罚单记录"
第二步：选择需要申诉的罚单
第三步：选择申诉理由类型：
  - 商家出餐延迟（需提供出餐时间截图）
  - 顾客地址不准确（需提供地址截图）
  - 交通管制/道路封锁（需提供导航截图）
  - 恶劣天气未自动延长（系统自动识别）
  - 其他原因（需补充说明）
第四步：上传相关证据材料（图片最多3张，单张不超过5MB）
第五步：提交申诉，等待审核

2.3 申诉审核时效
- 简单申诉：1-3个工作日
- 复杂申诉：3-5个工作日
- 申诉期间不执行罚款

2.4 申诉结果
- 申诉成功：全额退还罚款，并恢复骑手评分
- 申诉部分成功：按比例退还罚款
- 申诉失败：维持罚款，不可再次申诉同一罚单

第三章 收入与结算

3.1 配送费计算
配送费由基础配送费+距离配送费+时段补贴组成：
- 基础配送费：3-5元/单
- 距离配送费：每公里加收0.5-1元
- 时段补贴：高峰时段（11:00-13:00、17:00-19:00）额外补贴2元/单
- 恶劣天气补贴：恶劣天气期间每单额外补贴3-5元

3.2 提现规则
- 最低提现金额：50元
- 提现到账时间：
  - 银行卡：1-2个工作日
  - 支付宝：实时到账
  - 微信：实时到账
- 每日提现次数不限
- 提现手续费：0.1%，最低0.1元

3.3 奖励机制
- 完美送达奖励：连续100单无超时，奖励200元
- 高峰时段奖励：高峰时段完成订单数≥8单，额外奖励30元
- 好评奖励：获得用户5星好评，每单奖励1元
- 新骑手保护期：注册后前30天，超时罚款减半

第四章 安全规范

4.1 骑行安全要求
- 骑行时必须佩戴头盔
- 禁止逆行、闯红灯、违规变道
- 恶劣天气建议暂停接单
- 夜间配送建议穿着反光背心

4.2 食品安全要求
- 取餐时检查餐品包装完整性
- 热食类餐品必须使用保温箱
- 冷冻类餐品必须使用冰袋
- 禁止将餐品与个人物品混放
- 配送过程中禁止打开餐品包装

4.3 异常情况处理
遇到以下情况，骑手应立即上报：
- 餐品在配送途中损坏
- 顾客地址无法到达
- 遇到交通事故
- 餐品被动物污染
- 发现食品安全隐患

上报方式：APP首页"一键上报"按钮，选择异常类型，拍照上传证据。

第五章 骑手等级体系

5.1 等级划分
- 见习骑手：注册0-30天
- 初级骑手：注册30-90天
- 中级骑手：注册90-180天
- 高级骑手：注册180天以上
- 金牌骑手：前10%的中级及以上骑手

5.2 等级权益
- 见习骑手：基础配送费，接单范围2km
- 初级骑手：基础配送费×1.05，接单范围3km
- 中级骑手：基础配送费×1.1，接单范围5km
- 高级骑手：基础配送费×1.15，接单范围8km，优先派单
- 金牌骑手：基础配送费×1.2，接单范围10km，优先派单，专属客服
